# Front-End

Esta pasta contém projetos básicos e introdutórios desenvolvidos exclusivamente com as linguagens de front-end: **HTML**, **CSS** e **JavaScript**.

## Objetivo

Aqui você encontrará exemplos simples de aplicações web, interfaces responsivas e interações básicas com JavaScript. Estes projetos são ideais para quem está aprendendo ou buscando inspiração para desenvolver suas próprias páginas e aplicações web.

## Linguagens Utilizadas

- **HTML**: Estrutura das páginas web.
- **CSS**: Estilização e layout responsivo.
- **JavaScript**: Interatividade e lógica do lado cliente.

Todos os projetos nesta pasta utilizam apenas essas três tecnologias, sem frameworks ou bibliotecas externas, para manter o foco nos fundamentos.

## Projetos Incluídos

- `projetos html/`: Exemplos básicos de HTML/CSS/JS, como uma mini calculadora.
- `to-do-list/`: Uma aplicação de lista de tarefas simples com front-end em HTML/CSS/JS.

## Como Usar

Navegue pelas subpastas, abra os arquivos `.html` em um navegador para visualizar, e edite o código para aprender e experimentar!</content>
