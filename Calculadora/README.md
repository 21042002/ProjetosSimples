# Calculadora Simples

Este é um projeto de uma calculadora simples desenvolvida em Python utilizando a biblioteca CustomTkinter.

## Descrição

A calculadora permite realizar operações básicas de matemática: adição, subtração, multiplicação e divisão. Ela possui uma interface gráfica moderna com tema escuro por padrão.

## Pré-requisitos

- Python 3.6 ou superior instalado no sistema.

## Instalação da Biblioteca CustomTkinter

Para instalar a biblioteca CustomTkinter, execute o seguinte comando no terminal:

```bash
pip install customtkinter
```

Certifique-se de que o pip está atualizado. Se você estiver usando um ambiente virtual, ative-o antes de executar o comando.

## Como Iniciar o Projeto

1. Navegue até a pasta do projeto:
   ```bash
   cd ProjetosSimples/Calculadora
   ```

2. Execute o script Python:
   ```bash
   python Calculadora.py
   ```

A janela da calculadora será aberta e você poderá começar a usar.

## Diferenças entre CustomTkinter e Tkinter

- **Tkinter**: É a biblioteca padrão para criação de interfaces gráficas em Python. Ela é incluída na instalação padrão do Python e oferece widgets básicos. No entanto, a aparência é mais antiga e limitada em termos de personalização e temas modernos.

- **CustomTkinter**: É uma extensão moderna do Tkinter que fornece widgets com aparência mais atrativa e moderna. Inclui suporte nativo a temas dark e light, widgets mais estilizados (como botões arredondados, entradas com bordas suaves) e uma API mais intuitiva para personalização. CustomTkinter é ideal para aplicações que precisam de uma interface mais profissional e atual, sem sacrificar a simplicidade do Tkinter.

Em resumo, enquanto Tkinter é suficiente para aplicações básicas, CustomTkinter oferece uma experiência visual superior e é recomendado para projetos que valorizam a estética moderna.

## Funcionalidades

- Operações básicas: +, -, *, /
- Limpar display (botão C)
- Tratamento de erro para divisão por zero
- Interface responsiva com grid
